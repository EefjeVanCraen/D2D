AUTOMATIC BACKUP TO ONEDRIVE - QUICK SETUP
===========================================

OPTION 1: Manual Move (Easiest)
--------------------------------
1. When backups download to your Downloads folder
2. Double-click "move-backups-to-onedrive.bat"
3. Files are automatically moved to OneDrive backups folder
4. OneDrive syncs them to cloud automatically

OPTION 2: Automatic Scheduled Task (Advanced)
----------------------------------------------
1. Right-click "setup-auto-backup-mover.ps1"
2. Select "Run with PowerShell" (may need to run as Administrator)
3. This creates a task that moves backups every hour automatically

OPTION 3: Manual Backup with File Picker
-----------------------------------------
1. Go to Settings > Backup tab
2. Click "Create Backup Now"
3. When the file picker opens, navigate to:
   Day2Day Platform\backups\
4. Save the file there directly

RECOMMENDED: Use Option 1 - it's simple and works every time!
