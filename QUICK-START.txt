DAY2DAY PLATFORM - QUICK START GUIDE
=====================================

1. LAUNCH THE APPLICATION
   - Double-click "launch-day2day.bat" to open in your browser
   - Or open "index.html" directly

2. CREATE SHORTCUTS
   - Desktop: Run "create-desktop-shortcut.ps1" in PowerShell
   - Taskbar: Run "create-taskbar-shortcut.ps1" in PowerShell, then pin from Start Menu

3. FIRST TIME SETUP
   - Go to Settings (⚙️ icon in sidebar)
   - Add your departments (e.g., "Work", "Personal")
   - Add your tags (e.g., "Urgent", "Follow-up")
   - Your birthday (March 13) is already added

4. START USING
   - Add tasks in the Tasks section
   - Add ideas in the Ideas section
   - Add birthdays in the Birthdays section
   - Customize widgets on the Dashboard by dragging and resizing

5. WEEKLY SUMMARIES
   - Automatically created every Friday at 5 PM CET
   - View them in the Summaries section

6. DATA BACKUP
   - Automatic monthly backups are enabled by default
   - Create manual backups anytime from Settings > Backup tab
   - Export all data to Excel using "Export All" button in sidebar

7. RECYCLE BIN
   - Deleted items go to Recycle Bin
   - Restore or permanently delete from there

TIPS:
- Drag widgets to reposition them
- Resize widgets from the bottom-right corner
- Use filters to find specific tasks or ideas
- All data is saved automatically in your browser
